If you have forgotten your password: You can create a new password using security questions you previously set up. Visit: Click: Forgot Your Password? Fill out the requested information and click: Submit . Provide the answers to your security questions. Create a new password. If you do not remember your security questions, please contact the umTech Helpdesk at 678-8888 to have your password reset. Once this is done, you can create new security questions.

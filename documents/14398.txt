From blogs.memphis.edu , users can create and post content from any device that has an internet connection. Users can create a personal individual blog, or an interactive collaborative blog. Collaborative blogs allow users to come together and focus on a common subject such as a specific course. As with all other technology resources at U of M, blogs are also subject to the Acceptable Use of Information Technology Resources policy. Getting Started and Basic Training Posted in Facutly Resources , Student Resources Respondus LockDown Browser Posted on February 3, 2015 by Jessica Johnson The Respondus LockDown Browser is a tool used by some professors to help ensure academic integrity. When using the LockDown Browser, some features such as printing, going to another URL, and copying and pasting are disabled until the quiz is submitted. Before you can use the browser you will need to install it from the link within your eCourseware course. Once you have installed it, you can use the desktop icon to access elearn.memphis.edu through the LockDown Browser. Academic Integrity Student Quick Start Guide Posted in Student Resources Respondus StudyMate Posted on February 3, 2015 by Jessica Johnson StudyMate is a tool offered by Respondus that allows instructors to create self-assessments, games, and other learning activities. Instructors can choose to use either StudyMate Author, a Windows-based tool, or StudyMate Class, a web-based version. Each application has different features and abilities which allows instructors to choose the one that best suits their needs.

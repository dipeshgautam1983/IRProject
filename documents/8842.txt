Published on Desire2Learn Resource Center ( ) Home &gt; Quizzes &gt; Using Quizzes &gt; Creating bonus quiz questions Creating bonus quiz questions On the Manage Quizzes page, click on the quiz you want to create bonus questions for. Click Add/Edit Questions in the Properties tab. Select the check box beside each question you want to turn into bonus questions and click Edit Values . Select the Bonus check box for each question. Click Save . Note The bonus value is equal to the points value assigned to the quiz question. Audience: Instructor [1] Source URL: Links: [1]
